# LeetCode 1304. Find N Unique Integers Sum up to Zero

## Problem Statement
Given an integer `n`, return **any** array containing `n` **unique** integers such that they sum up to `0`.

## Input
- The input consists of a single line containing an integer `n`.

## Output
- Print `n` space-separated unique integers whose sum is equal to 0.

## Constraints
- `1 <= n <= 1000`

## Examples

### Example 1
Input:
5

Output:
-2 -1 0 1 2

**Explanation:** An array containing `[-2, -1, 0, 1, 2]` has a sum of 0, and all elements are unique. Other valid answers include `[-10, -5, 0, 5, 10]`.

### Example 2
Input:
3

Output:
-1 0 1

### Example 3
Input:
1

Output:
0
