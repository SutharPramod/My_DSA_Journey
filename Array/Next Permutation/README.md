# 31. Next Permutation

## ðŸ“ Problem Description
A **permutation** of an array of integers is an arrangement of its members into a sequence or linear order. The **next permutation** is the next lexicographically greater arrangement.

If such an arrangement is not possible (i.e., the array is in descending order), the array must be rearranged as the **lowest possible order** (sorted in ascending order).
